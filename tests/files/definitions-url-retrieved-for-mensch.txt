          <span class="etymwb-headword">

            <span class="etymwb-gramgrp">

höchstentwickeltes gesellschaftliches Lebewesen
mit der Fähigkeit zu arbeiten und zu denken,
<abbr title="althochdeutsch" data-toggle="tooltip">ahd.</abbr>
<span class="etymwb-mentioned">mennisco</span>

<span class="etymwb-mentioned">mensch</span>

<abbr title="mittelniederdeutsch" data-toggle="tooltip">mnd.</abbr>
<span class="etymwb-mentioned">minsche</span>,
<abbr title="mittelniederländisch" data-toggle="tooltip">mnl.</abbr>
<span class="etymwb-mentioned">mensce</span>,
<abbr title="niederländisch" data-toggle="tooltip">nl.</abbr>

(um 800),
<abbr title="mittelhochdeutsch" data-toggle="tooltip">mhd.</abbr>
<span class="etymwb-mentioned">mennisch</span>

<abbr title="altsächsisch" data-toggle="tooltip">asächs.</abbr>
<span class="etymwb-mentioned">mennisk</span>,
<abbr title="altenglisch" data-toggle="tooltip">aengl.</abbr>
<span class="etymwb-mentioned">mennisc</span>,
<abbr title="altnordisch" data-toggle="tooltip">anord.</abbr>
<span class="etymwb-mentioned">mennskr</span>,
<abbr title="gotisch" data-toggle="tooltip">got.</abbr>
<span class="etymwb-mentioned">mannisks</span>
vorliegenden Adjektivs
<abbr title="germanisch" data-toggle="tooltip">germ.</abbr>
<span class="etymwb-mentioned">*manniska-</span>

einer Ableitung von dem unter
<i class="bi bi-arrow-up-right dwds-aur"></i><a href="/wb/etymwb/Mann" class="intern">Mann</a>
(s. d.)

Das im <abbr title="mittelhochdeutsch" data-toggle="tooltip">Mhd.</abbr> auftretende Neutrum

bezeichnet jedoch seit dem 15. Jh. häufig eine
‘weibliche Person’,
besonders die
‘Magd’,
und wird seit dem 18. Jh. durchgehend abwertend gebraucht;
dazu auch der Plural
<span class="etymwb-mentioned">Menscher</span>
(17. Jh.).

  <h2 id="etymwb-2">Etymologie</h2>
  <div style="position:relative" data-track-content data-content-name="DWDSWB Artikel" data-content-piece="Etymologie">
    <div class="glyphicon glyphicon-chevron-down btn-link" style="cursor:pointer;position:absolute;left:-25px;top:5px; display:none" aria-hidden="true" data-toggle="tooltip" title="Etymologie aufklappen"></div>
    
      <div style="position:relative">
        

        

        
          <div style="font-family:'Source Sans Pro',sans-serif;font-size:10pt;color:#979797">
            Mensch · menschlich · Menschlichkeit · Menschheit · Menschenfeind · Menschenfresser · menschenmöglich · Menschenrecht · Menschentum · Übermensch · übermenschlich · Unmensch · Untermensch
          </div>
        
        <div class="etymwb-wrapper"><div class="etymwb-entry">
          <span class="etymwb-headword">
            <span class="etymwb-orth">Mensch</span>
            <span class="etymwb-gramgrp">
              
              <abbr data-toggle="tooltip" title="Maskulinum">m.</abbr>
            </span>
          </span>
          
höchstentwickeltes gesellschaftliches Lebewesen
mit der Fähigkeit zu arbeiten und zu denken,
<abbr title="althochdeutsch" data-toggle="tooltip">ahd.</abbr>
<span class="etymwb-mentioned">mennisco</span>
m.
‘Mensch’
(8. Jh.),
<abbr title="mittelhochdeutsch" data-toggle="tooltip">mhd.</abbr>
<span class="etymwb-mentioned">mensche</span>,
<span class="etymwb-mentioned">mensch</span>
m. n.
‘Mensch, Mädchen, Buhlerin, Magd, Knecht, das menschliche Geschlecht’,
<abbr title="mittelniederdeutsch" data-toggle="tooltip">mnd.</abbr>
<span class="etymwb-mentioned">minsche</span>,
<abbr title="mittelniederländisch" data-toggle="tooltip">mnl.</abbr>
<span class="etymwb-mentioned">mensce</span>,
<abbr title="niederländisch" data-toggle="tooltip">nl.</abbr>
<span class="etymwb-mentioned">mens</span>
sind aus einer Substantivierung des in
<abbr title="althochdeutsch" data-toggle="tooltip">ahd.</abbr>
<span class="etymwb-mentioned">mennisc</span>
(um 800),
<abbr title="mittelhochdeutsch" data-toggle="tooltip">mhd.</abbr>
<span class="etymwb-mentioned">mennisch</span>
‘menschlich, mannhaft’,
<abbr title="altsächsisch" data-toggle="tooltip">asächs.</abbr>
<span class="etymwb-mentioned">mennisk</span>,
<abbr title="altenglisch" data-toggle="tooltip">aengl.</abbr>
<span class="etymwb-mentioned">mennisc</span>,
<abbr title="altnordisch" data-toggle="tooltip">anord.</abbr>
<span class="etymwb-mentioned">mennskr</span>,
<abbr title="gotisch" data-toggle="tooltip">got.</abbr>
<span class="etymwb-mentioned">mannisks</span>
vorliegenden Adjektivs
<abbr title="germanisch" data-toggle="tooltip">germ.</abbr>
<span class="etymwb-mentioned">*manniska-</span>
hervorgegangen,
einer Ableitung von dem unter
<i class="bi bi-arrow-up-right dwds-aur"></i><a href="/wb/etymwb/Mann" class="intern">Mann</a>
(s. d.)
behandelten Substantiv.
Das im <abbr title="mittelhochdeutsch" data-toggle="tooltip">Mhd.</abbr> auftretende Neutrum
steht bis ins 17. Jh. ohne abschätzigen Sinn neben dem Maskulinum,
bezeichnet jedoch seit dem 15. Jh. häufig eine
‘weibliche Person’,
besonders die
‘Magd’,
und wird seit dem 18. Jh. durchgehend abwertend gebraucht;
dazu auch der Plural
<span class="etymwb-mentioned">Menscher</span>
(17. Jh.).
 – 
<span class="etymwb-headword">
  <span class="etymwb-orth">menschlich</span>
  <span class="etymwb-gramgrp">
    Adj.
  </span>
</span>
‘den Menschen betreffend, ihm gemäß, hilfsbereit, gut’,
<abbr title="althochdeutsch" data-toggle="tooltip">ahd.</abbr>
<span class="etymwb-mentioned">mennisclīh</span>
(8. Jh.),
<abbr title="mittelhochdeutsch" data-toggle="tooltip">mhd.</abbr>
<span class="etymwb-mentioned">menschlich</span>
‘nach, von Menschenart’;
dazu
<span class="etymwb-headword">
  <span class="etymwb-orth">Menschlichkeit</span>
  <span class="etymwb-gramgrp">
    
    <abbr data-toggle="tooltip" title="Femininum">f.</abbr>
  </span>
</span>
‘Menschennatur, menschliche Würde, menschliches Gefühl, Freundlichkeit im Umgang mit anderen’
(16. Jh.,
Übersetzung von
<abbr title="lateinisch" data-toggle="tooltip">lat.</abbr>
<span class="etymwb-mentioned">hūmānitās</span>,
s.
<i class="bi bi-arrow-up-right dwds-aur"></i><a href="/wb/etymwb/Humanit%C3%A4t" class="intern">Humanität</a>),
geläufig seit dem 18. Jh.,
vereinzelt
‘Menschheit’
(nur 17. Jh.);
vgl.
<abbr title="spätmittelhochdeutsch" data-toggle="tooltip">spätmhd.</abbr>
<span class="etymwb-mentioned">menschlīcheit</span>
‘Zustand des Menschseins, Menschheit’.
<span class="etymwb-headword">
  <span class="etymwb-orth">Menschheit</span>
  <span class="etymwb-gramgrp">
    
    <abbr data-toggle="tooltip" title="Femininum">f.</abbr>
  </span>
</span>
‘Gesamtheit der Menschen’,
<abbr title="althochdeutsch" data-toggle="tooltip">ahd.</abbr>
<span class="etymwb-mentioned">mennischeit</span>
‘alle Menschen, menschliche Natur, Menschwerdung’
(um 1000),
<abbr title="mittelhochdeutsch" data-toggle="tooltip">mhd.</abbr>
<span class="etymwb-mentioned">menschheit</span>,
<span class="etymwb-mentioned">menscheit</span>
‘alle Menschen, Natur und Leben eines Menschen, Menschlichkeit’.
<span class="etymwb-headword">
  <span class="etymwb-orth">Menschenfeind</span>
  <span class="etymwb-gramgrp">
    
    <abbr data-toggle="tooltip" title="Maskulinum">m.</abbr>
  </span>
</span>
‘Menschenhasser, -verächter’
(16. Jh.),
Übersetzung von
<abbr title="griechisch-lateinisch" data-toggle="tooltip">griech.-lat.</abbr>
<span class="etymwb-mentioned">misanthropus</span>
(s.
<i class="bi bi-arrow-up-right dwds-aur"></i><a href="/wb/etymwb/Misanthrop" class="intern">Misanthrop</a>).
<span class="etymwb-headword">
  <span class="etymwb-orth">Menschenfresser</span>
  <span class="etymwb-gramgrp">
    
    <abbr data-toggle="tooltip" title="Maskulinum">m.</abbr>
  </span>
</span>
‘Kannibale’,
Übersetzung (17. Jh.) von gleichbed.
<abbr title="lateinisch" data-toggle="tooltip">lat.</abbr>
<span class="etymwb-mentioned">anthrōpophagus</span>,
<abbr title="griechisch" data-toggle="tooltip">griech.</abbr>
<span class="etymwb-mentioned">anthrōpophágos</span>
(<span class="etymwb-mentioned">ἀνθρωποφάγος</span>);
vgl.
<abbr title="griechisch" data-toggle="tooltip">griech.</abbr>
<span class="etymwb-mentioned">ánthrōpos</span>
(<span class="etymwb-mentioned">ἄνθρωπος</span>)
‘Mensch’
und
<span class="etymwb-mentioned">phágos</span>
(<span class="etymwb-mentioned">φάγος</span>)
‘Fresser’,
<span class="etymwb-mentioned">phagé͞in</span>
(<span class="etymwb-mentioned">φαγεῖν</span>)
‘essen, verzehren’.
<span class="etymwb-headword">
  <span class="etymwb-orth">menschenmöglich</span>
  <span class="etymwb-gramgrp">
    Adj.
  </span>
</span>
‘für einen Menschen möglich’
(18. Jh.),
aus älterem
<span class="etymwb-mentioned">menschmöglich</span>
(17. Jh.),
das aus
<span class="etymwb-mentioned">menschlich</span>
und
<span class="etymwb-mentioned">möglich</span>
(16. Jh.)
mit Ersparung der Suffixwiederholung zusammengerückt ist.
<span class="etymwb-headword">
  <span class="etymwb-orth">Menschenrecht</span>
  <span class="etymwb-gramgrp">
    
    <abbr data-toggle="tooltip" title="Neutrum">n.</abbr>
  </span>
</span>
zunächst
‘von Menschen eingesetztes Recht’
(Ende 17. Jh.);
<span class="etymwb-mentioned">Menschheitsrechte</span>
Plur.
‘dem Menschen von Geburt an zukommende Rechte und Würde’
(nach 1750).
Danach
(meist im Plur.)
<span class="etymwb-mentioned">Menschenrechte</span>
‘Reihe von Rechten, die die Stellung des Menschen in der Gesellschaft und sein Verhältnis zum Staat regeln’
(seit 1777).
Durch die amerikanische Unabhängigkeitserklärung
(1776)
und die französische
Déclaration des droits de l’homme et du citoyen
(1789)
wird der Begriff zum politischen Schlagwort
und mit seinem damaligen bürgerlichen Inhalt
(Freiheit, Gleichheit, Brüderlichkeit,
Recht auf Eigentum und Sicherheit
sowie Widerstand gegen Unterdrückung)
auch im <abbr title="deutsch" data-toggle="tooltip">Dt.</abbr> geläufig;
wahrscheinlich geht auch die vorherrschende Verwendung im Plural
auf das <abbr title="französisch" data-toggle="tooltip">frz.</abbr> Vorbild zurück.
<span class="etymwb-headword">
  <span class="etymwb-orth">Menschentum</span>
  <span class="etymwb-gramgrp">
    
    <abbr data-toggle="tooltip" title="Neutrum">n.</abbr>
  </span>
</span>
‘Wesen des Menschen, das Dasein als Mensch’
(Anfang 19. Jh.),
zuvor
‘Menschheit’
(17. Jh.).
<span class="etymwb-headword">
  <span class="etymwb-orth">Übermensch</span>
  <span class="etymwb-gramgrp">
    
    <abbr data-toggle="tooltip" title="Maskulinum">m.</abbr>
  </span>
</span>
in der Theologie gebildet (16. Jh.) aus
<span class="etymwb-headword">
  <span class="etymwb-orth">übermenschlich</span>
  <span class="etymwb-gramgrp">
    Adj.
  </span>
</span>
‘in jeder Weise über das durchschnittliche menschliche Vermögen hinausragend’
(Anfang 16. Jh.)
zur Bezeichnung eines
‘moralisch überdurchschnittlichen Menschen, eines übermenschlichen Wesens’
(so noch im 18./19. Jh.);
bei
<a href="https://de.wikipedia.org/wiki/Johann_Gottfried_Herder">Herder</a>
im Sinne von
‘Heros, Halbgott, Genie’;
bei
<a href="https://de.wikipedia.org/wiki/Friedrich_Nietzsche">Nietzsche</a>
(Zarathustra)
im Gegensatz zum Durchschnittsmenschen
‘derjenige, der den Menschen überwunden hat’.
Daran anknüpfend wird
<span class="etymwb-mentioned">Übermensch</span>
(wie
<span class="etymwb-mentioned">Herrenmensch</span>,
Gegensatz
<span class="etymwb-mentioned">Untermensch</span>,
s. unten)
zum Schlagwort der faschistischen Rassenideologie.
<span class="etymwb-headword">
  <span class="etymwb-orth">Unmensch</span>
  <span class="etymwb-gramgrp">
    
    <abbr data-toggle="tooltip" title="Maskulinum">m.</abbr>
  </span>
</span>
‘brutaler, grausamer Mensch’,
<abbr title="althochdeutsch" data-toggle="tooltip">ahd.</abbr>
<span class="etymwb-mentioned">unmennisco</span>
‘Nichtmensch’
(um 1000),
<abbr title="spätmittelhochdeutsch" data-toggle="tooltip">spätmhd.</abbr>
<span class="etymwb-mentioned">unmensch</span>
‘was nicht den Namen Mensch verdient, Bösewicht’,
<abbr title="frühneuhochdeutsch" data-toggle="tooltip">frühnhd.</abbr>
‘Bestie, Kannibale, Zauberer’
(16. Jh.),
‘Menschenfeind’
(18. Jh.).
<span class="etymwb-headword">
  <span class="etymwb-orth">Untermensch</span>
  <span class="etymwb-gramgrp">
    
    <abbr data-toggle="tooltip" title="Maskulinum">m.</abbr>
  </span>
</span>
‘niedriges Wesen, das sittlich unterhalb der menschlichen Stufe steht’
(18. Jh.);
danach Schlagwort der Faschisten
für die Opfer ihrer Unterdrückungs- und Ausrottungspolitik.
